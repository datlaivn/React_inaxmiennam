# SQHOME — React scaffold

This is a minimal Vite + React scaffold for the SQHOME project.

What's included:
- Vite setup (dev/build/preview)
- `react-router-dom` with `Header` component pre-made using your provided HTML (converted to JSX)
- Shared CSS at `src/index.css`
- `Home` and `Footer` components left intentionally empty for you to fill

How to run:

```bash
cd "/Users/datlight/Desktop/Code Shop/1_SQHOME"
npm install
npm run dev
```

Files you may want to edit first:
- `src/components/Header.jsx` — header/nav JSX
- `src/index.css` — shared styles
- `src/components/Home.jsx` — homepage content
- `src/components/Footer.jsx` — footer content
